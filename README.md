Project Title
To Do App: CSS Flexbox Layout

Brief Project Description
This project is a static, responsive web interface for a daily To-Do application. It demonstrates the practical application of modern CSS Flexbox properties to create dynamic layouts. Key concepts implemented include:

Using flex and inline-flex to structure main containers.

Utilizing flex-grow to allocate proportional space between the week schedule and the reminders sidebar.

Applying flex-wrap: wrap to ensure the daily task squares gracefully drop to the next line on smaller screens while keeping their sequential order intact.

Using justify-content and align-items to perfectly center content both horizontally and vertically.

Setup / Run Instructions
Save both index.html and style.css into the same folder on your computer.

Ensure you have an active internet connection (the project imports fonts from Google Fonts: 'Quicksand' and 'Rock Salt').

Open the index.html file using any modern web browser (Google Chrome is recommended).

To test the responsiveness, slowly resize your browser window horizontally to watch the task squares wrap to the next line.
